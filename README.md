# Json MegaApp

Desktop application built with Electron for viewing, editing, and manipulating JSON files.

## Features

- **JSON Editor**: Open JSON files and browse their structure in a collapsible tree. Edit values directly from the interface.
- **Replacer**: Define keys (using JSONPath syntax) and replacement values, select multiple destination files, and apply changes in bulk.
- **Configurations**: Save and load replacer configurations to reuse across different projects.

## Usage

```bash
npm run dev
```

# Json MegaApp

Aplicación de escritorio hecha con Electron para visualizar, editar y manipular archivos JSON.

## Funcionalidades

- **Editor JSON**: Abre archivos JSON y navegá por su estructura en un árbol colapsable. Editá los valores directamente desde la interfaz.
- **Remplazador**: Definí claves (usando sintaxis JSONPath) y valores a reemplazar, seleccioná múltiples archivos destino, y aplicá los cambios de forma masiva.
- **Configuraciones**: Guardá y cargá configuraciones del remplazador para reutilizar en distintos proyectos.

## Uso

```bash
npm run dev
```
